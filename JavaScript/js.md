#### Array

- ##### some()

  ```javascript
  //some()方法测试数组中是不是至少有1个元素通过了被提供的函数测试。它返回的是一个Boolean类型的值
  [1,2,3].some(i => i === 2) // true
  ```

- ##### concat()

  ```javascript
  //concat()方法适用于合并2个或多个数组。此方法不会改变现有数组，而是返回一个新的数组
  [1,2,3].concat(['jok']) //[1,2,3,'jok']
```
